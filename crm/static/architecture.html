Service Integration Map

Bidirectional PCM audio stream. Handles intent detection, voice transcription, tool dispatch. Agent sees analysis results, not raw images.

Gemini 3 Pro: inpainting on photos + style previews. Imagen 4 Ultra: fresh room generation fallback. Images stored in GCS with 7d TTL.
